# ADMIN KEY LOCATION
# DO NOT COMMIT THIS FILE TO GIT

Your admin key is stored in:
  .env.docker

To view your key:
  grep "^ADMIN_KEY=" .env.docker

To rotate your key:
  NEW_KEY=$(openssl rand -hex 32)
  sed -i.bak "s/^ADMIN_KEY=.*/ADMIN_KEY=${NEW_KEY}/" .env.docker
  docker compose restart api

SECURITY NOTES:
- Never share your admin key
- Never commit .env.docker to git
- Rotate keys every 90 days
- Use HTTPS in production
- Enable SECURE_COOKIES=true after HTTPS
